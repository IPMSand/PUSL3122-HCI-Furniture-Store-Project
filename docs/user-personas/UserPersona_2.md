## Background

Mohammed is renovating his home and wants to furnish multiple rooms. He expects guidance from professionals and feels uncomfortable when exposed to complex 3D software. He prefers either a very simple interface or observing the designer operate the tool while he provides feedback.

---

## Goals

* Select pre-set room templates
* See realistic shading and room visualization
* Receive confirmation before deleting items
* View loading animation with time estimate
* Read measurements in large, clear fonts
* Access a shareable viewer link
* Observe while salesperson controls the tool

---

## Frustrations / Pain Points

* Feels lost using complex 3D tools
* Fear of making mistakes
* Small text and unclear measurements
* Overwhelming number of buttons

---

## Key Non-Functional Needs

* Beginner-friendly interface
* Accessibility (large fonts)
* Clear feedback and loading indicators
* Simple navigation structure
