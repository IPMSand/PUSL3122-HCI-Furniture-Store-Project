# User Stories — Youth (Sara)

---

### US-Y1

**Role:** As Sara, a student / hobbyist,

**Action:** I want to drag walls on a finger-friendly grid

**Benefit:** So that I can set up my room shape without typing any dimensions.

**Acceptance Criteria:**

* Wall segments respond to touch/drag on a visible grid.
* Room outline updates in real time.

---

### US-Y2

**Role:** As Sara,

**Action:** I want to apply a full-room color theme (e.g., Modern Blue) in one tap

**Benefit:** So that I can quickly visualize different styles.

**Acceptance Criteria:**

* 'Global Theme' panel shows preset color schemes.
* Selecting one applies matching colors to all furniture and walls instantly.

---

### US-Y3

**Role:** As Sara,

**Action:** I want a clearly visible Undo button

**Benefit:** So that I can fix mistakes without searching through menus.

**Acceptance Criteria:**

* Undo button permanently visible in toolbar.
* Tapping reverses the last action with a confirmation animation.

---

### US-Y4

**Role:** As Sara,

**Action:** I want to share my design as a viewer link

**Benefit:** So that I can send it to friends on messaging apps.

**Acceptance Criteria:**

* 'Share' button generates a short link.
* Link opens read-only 3D view in any browser without login.
