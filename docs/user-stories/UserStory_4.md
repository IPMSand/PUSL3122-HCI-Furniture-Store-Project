# User Stories — Elderly (Kamal)

---

### US-E1

**Role:** As Kamal, a retired homeowner,

**Action:** I want to enter room dimensions by typing numbers

**Benefit:** So that I can use the exact measurements I already know from measuring my bedroom.

**Acceptance Criteria:**

* Simple numeric input fields accept dimensions in meters.
* Room is drawn automatically with a labeled diagram.

---

### US-E2

**Role:** As Kamal,

**Action:** I want all measurement labels and menu text displayed in a large, readable font

**Benefit:** So that I don't have to strain to read the screen.

**Acceptance Criteria:**

* All text defaults to minimum 14pt.
* 'Increase Font Size' toggle visible on the home screen.

---

### US-E3

**Role:** As Kamal,

**Action:** I want a high-contrast display mode

**Benefit:** So that I can clearly distinguish buttons and furniture labels.

**Acceptance Criteria:**

* High-contrast toggle available.
* Enabling it switches to dark-background / white-text / bold-icon theme.

---

### US-E4

**Role:** As Kamal,

**Action:** I want a loading animation with a time estimate

**Benefit:** So that I know the design is still being prepared and I haven't lost my work.

**Acceptance Criteria:**

* 3D model loading shows spinning animation with countdown (e.g., "About 15 seconds") in the center of the screen.
