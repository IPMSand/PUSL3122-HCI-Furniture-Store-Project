# User Stories — Customer (Mohammed)

---

### US-C1

**Role:** As Mohammed, a homeowner,

**Action:** I want to select a pre-set room template (L-shape, square, rectangle)

**Benefit:** So that I don't have to configure the room shape manually.

**Acceptance Criteria:**

* Gallery of templates is shown at the start.
* Selecting a template instantly creates the room outline.
* No further input needed.

---

### US-C2

**Role:** As Mohammed,

**Action:** I want the salesperson to control the design screen while I watch

**Benefit:** So that I can focus on giving my opinion without feeling overwhelmed.

**Acceptance Criteria:**

* Interface supports 'presenter mode'.
* One person navigates while the client observes.
* Mirrored or shared screen works without extra login.

---

### US-C3

**Role:** As Mohammed,

**Action:** I want a pop-up confirmation before anything is deleted

**Benefit:** So that I don't lose my design by accident.

**Acceptance Criteria:**

* Every delete triggers a confirmation dialog.
* Cancel and Confirm buttons available.
* Undo option is also available if deletion occurs.

---

### US-C4

**Role:** As Mohammed,

**Action:** I want to receive a viewer link after the session

**Benefit:** So that I can review the 3D design with my family at home before making a purchase.

**Acceptance Criteria:**

* Shareable link generated from completed design.
* Opens read-only 3D view.
* No login required.
