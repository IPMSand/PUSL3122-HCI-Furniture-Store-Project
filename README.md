## Use 

  - `docs` for analysis
  - `design` for designs
  - `app` for development
  - `tests` for testing
## Initial Structure for now
```text
/furniture-hci-project
├── /docs            # Requirements & Analysis (Maintainer Lead)
│   ├── /research    # Survey results & raw data
│   ├── /personas    # User Persona profiles
│   ├── /requirements # User Stories & Functional Specs
│   └── /hci-theory   # Competitive analysis & accessibility
├── /design          # UI/UX Area (Designer Lead)
│   ├── /wireframes   # Low-fi sketches
│   ├── /prototypes   # Figma links & mockups
│   └── /assets       # Logos, icons, and colors
├── /app             # Codebase (Developer Lead)
│   ├── /src         # Logic, components, and 3D rendering
│   └── package.json # Project dependencies
└── /tests           # Evaluation Area
    ├── /usability   # User feedback & scripts
    └── /unit        # Code testing
 ```
